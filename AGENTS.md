\# AGENTS.md



\## Project

Diploma project: software for monitoring trends in the regional IT labor market.



\## Product idea

This is an analytics and monitoring web application, not a job board.



\## Primary domain scope

\- Primary region: Novosibirsk Oblast

\- Future scaling to other Russian regions must be possible



\## Tech stack

\- Frontend: React + TypeScript

\- Backend: FastAPI

\- Database: PostgreSQL

\- ORM: SQLAlchemy

\- Migrations: Alembic

\- Styling: Tailwind CSS

\- Charts: Recharts

\- Architecture: monolithic web application



\## Coding principles

\- Keep the code simple and readable

\- Avoid overengineering

\- Prefer clean folder structure

\- Write code suitable for a student diploma project

\- Add comments only where really useful

\- Make all modules easy to explain in the thesis



\## Current stage

We are building the initial MVP scaffold with mock data only.
